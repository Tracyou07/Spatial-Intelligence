# VGGT — Visual Geometry Grounded Transformer

## 核心思想
**单次前向推理**，从 1 张到数百张图像中直接预测场景所有关键 3D 属性：
相机内外参、点云、深度图、3D 点轨迹。无需迭代优化。

## 架构
```
N 张图像（任意数量）
  → Patch Embedding + 位置编码
  → Aggregator（Transformer，跨帧交互）
  → 多个并行 Head：
      ├→ camera_head  → 相机外参 (R,t) + 内参 (f, cx, cy)
      ├→ depth_head   → 深度图 + 置信度
      ├→ point_head   → 3D pointmap + 置信度
      └→ track_head   → 3D 点轨迹（跨帧跟踪）
```

## 关键点
- **1B 参数**，从 HuggingFace 加载：`facebook/VGGT-1B`
- 支持 1 张到数百张图，灵活输入
- 同时输出：相机位姿 + 深度 + 点云 + 轨迹，四合一
- 比 depth_head 更精准的做法：用 depth + camera 反投影得到点云
- 不需要特定顺序，视图之间通过 Aggregator 全局交互
- 输出可直接存 COLMAP 格式，接 gsplat / NeRF / 3DGS

## 使用
```python
from vggt.models.vggt import VGGT
from vggt.utils.load_fn import load_and_preprocess_images

model = VGGT.from_pretrained("facebook/VGGT-1B").to(device)
images = load_and_preprocess_images(["a.png", "b.png"]).to(device)

with torch.no_grad():
    predictions = model(images)
    # predictions 包含 cameras, depth_map, point_map, tracks
```

## 与同类方法对比
| | DUSt3R | MASt3R | CUT3R | VGGT |
|--|--|--|--|--|
| 输入 | 图像对 | 图像对 | 视频流（在线） | 任意多视图 |
| 输出 | pointmap | pointmap+特征 | pointmap+位姿 | 位姿+深度+点云+轨迹 |
| 后处理 | 需 BA | 需 SfM | 无 | 无 |
| 规模 | 小 | 中 | 在线 | 1~数百帧 |
| 速度 | 慢（BA） | 慢 | 实时 | 秒级 |
