# MUST3R — Multi-view Network for Stereo 3D Reconstruction

## 核心思想
将 DUSt3R/MASt3R 从"图像对"扩展到**真正的多视图**，引入多层记忆机制避免 O(N²) 的图像对复杂度，支持千帧级在线重建。

## 架构
```
Images {I1, ..., IN}
  → Symmetric Encoder（不区分参考帧/目标帧）
  → Multi-layer Memory（存储历史帧信息）← 核心创新
  → Decoder
  → 直接在公共坐标系下输出所有视图的 3D pointmap + 相机位姿
```

## 关键点
- **对称架构**：DUSt3R 有参考帧和目标帧之分，MUST3R 消除了这一区别
- **多层 Memory**：O(N) 复杂度替代 O(N²)，可扩展到大规模图像集
- 支持**在线 + 离线**两种模式
- 直接在统一坐标系下输出，无需后期全局对齐
- 安装：`pip install must3r@git+https://github.com/naver/must3r.git`

## 与前辈的对比
| | DUSt3R | MASt3R | MUST3R |
|--|--|--|--|
| 输入 | 图像对 | 图像对 | 多视图 |
| 复杂度 | O(N²) | O(N²) | O(N) |
| 特征匹配 | 无 | 有 | 有 |
| 坐标系 | 需全局对齐 | 需全局对齐 | 直接统一 |
| 规模 | 小 | 中 | 千帧级 |
