# MASt3R — Grounding Image Matching in 3D

## 核心思想
在 DUSt3R 基础上增加**局部特征描述子头**，把 3D 重建和图像特征匹配统一到一个模型中。

## 架构
```
I1, I2
  → Siamese ViT Encoder（self-attention）
  → Transformer Decoder（cross-attention）
  ├→ 3D Head   → X1, X2（dense pointmap）    ← 继承自 DUSt3R
  └→ Desc Head → D1, D2（dense 特征描述子）  ← 新增
  → Fast Reciprocal Matching（快速互匹配）
```

## 关键点
- 比 DUSt3R 多了 `Desc Head`，同时输出局部特征
- 特征用于**快速图像匹配**，大幅提升多视图重建精度
- **MASt3R-SfM**：基于此做大规模 Structure-from-Motion
- JOSH 的场景重建分支直接使用 MASt3R-SfM

## 与 DUSt3R 的区别
| | DUSt3R | MASt3R |
|--|--|--|
| 输出 | pointmap + confidence | pointmap + confidence + 描述子 |
| 匹配 | 无 | Fast Reciprocal Matching |
| SfM | 需 BA | MASt3R-SfM |
