# JOSH — Joint Optimization for 4D Human-Scene Reconstruction in the Wild

## 核心思想
**联合优化**场景几何和人体全局运动，以人体-场景**接触约束**作为桥梁，解决两者的尺度对齐问题。

## Pipeline
```
Video Frames
  ├→ Scene Branch: MASt3R-SfM
  │     → Dense pointmaps Xt
  │     → Camera poses Pt
  │     → Scale σt
  └→ Human Branch: Local HMR
        → Local SMPL pose θt
        → Local-to-camera transform Tc
        → SMPL meshes（camera frame）
  → Contact Correspondence
        → visible 人体顶点投影到场景点
        → 构建接触集合 Dt, Et
  → Joint Optimization
        Loss = L3D + L2D + Lc1(contact) + Lc2(temporal smooth) + Lp(pose prior)
  → Global SMPL + Aligned Scene + Camera Poses
```

## 关键点
- **非端到端**：先分别初始化场景和人体，再联合优化
- 核心创新：**接触约束**将人体顶点和场景点云绑定
- 支持 in-the-wild 网络视频
- 场景重建用 MASt3R-SfM，人体用 HMR

## 与 Human3R 的区别
| | JOSH | Human3R |
|--|--|--|
| 方式 | 优化 pipeline | 端到端前向 |
| 人场关系 | 显式接触约束 | 隐式特征融合 |
| 适用场景 | 高精度离线处理 | 实时在线 |
