# DPT — Vision Transformers for Dense Prediction

## 核心思想
用 ViT 替代 CNN backbone 做 dense prediction（深度估计、语义分割）。
ViT 天然具有全局感受野，弥补 CNN 局部卷积的短板。

## 架构
```
Input Image
  → ViT Encoder（patch tokens，全局感受野）
  → Reassemble Blocks（tokens → feature map）
  → Fusion Blocks（类 RefineNet，逐层上采样融合）
  → Prediction Head（dense depth / segmentation map）
```

## 关键点
- ViT 本身"空间无关"，靠**位置编码 + Self-Attention** 学习空间关系
- Self-Attention 让每个 patch 与所有 patch 交互 → 全局空间理解
- 端到端训练
- 输出是**深度图**，不是 3D 点云

## 局限
- 只做单图深度估计，没有跨视图的 3D 重建能力
- 引出问题：如何获得真正的 3D 空间信息？
