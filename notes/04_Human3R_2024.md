# Human3R — Everyone Everywhere All at Once

## 核心思想
在 CUT3R 框架上融合人体重建，**单次前向推理**同时输出：场景 3D + 多人 SMPL-X + 相机轨迹。

## 架构
```
Input Frame
  ├→ CUT3R ViT Encoder     （场景分支）
  └→ Multi-HMR ViT Encoder （人体分支）
  → State Interaction with HMR（人场融合，继承自 CUT3R 状态机制）
  ├→ Head_cam / Head_world → 场景点云（camera / world 坐标）
  ├→ Head_pose             → 相机外参 [R|t]
  └→ Human Branch          → SMPL-X（joints, pose, shape, mesh）
```

## 关键点
- **双编码器**：场景用 CUT3R ViT，人体用 Multi-HMR ViT
- 支持**多人**同时重建（everyone）
- 覆盖任意场景（everywhere）
- 单次前向（all at once），15 FPS，8GB 显存
- 在 BEDLAM 合成数据集上训练 1 天

## 与 JOSH 的区别
| | Human3R | JOSH |
|--|--|--|
| 方式 | 端到端前向 | 分步联合优化 |
| 速度 | 实时 15 FPS | 慢（优化迭代） |
| 人场交互 | 隐式（特征融合） | 显式（接触约束） |
