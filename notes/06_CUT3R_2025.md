# CUT3R — Continuous 3D Perception Model with Persistent State

## 核心思想
在线视频流中维护一个**持续更新的全局状态**，每帧输入后更新状态，支持推断未观测区域，直接输出相机位姿。

## 架构
```
Image_t → ViT Encoder → Tokens [Ft, Zt]
  → concatenate[Ft, Zt]
  → State Interaction Module  ← 核心创新：与历史状态交互
  ├→ Head_cam   → camera-space 3D pointmap
  ├→ Head_pose  → 相机外参 [R|t]
  └→ Head_world → world-space 3D pointmap
  → 更新全局状态 State_{t+1}
```

## 关键点
- **Persistent State**：维护全局场景记忆，跨帧持续更新
- 同时输出相机坐标系和世界坐标系的点云
- 直接预测相机外参，无需后处理 BA
- 支持推断**未被观测区域**（virtual view synthesis）
- 在线处理，适合视频流场景

## 与 MUST3R 的区别
| | CUT3R | MUST3R |
|--|--|--|
| 状态机制 | Persistent State（RNN-like） | Multi-layer Memory |
| 位姿预测 | 直接输出 | 需额外处理 |
| 适用 | 视频流（在线） | 图像集（在线+离线） |

## 衍生工作
- **Human3R**：在此框架上加入人体编码器
- **TTT3R**：在此基础上做 test-time training，提升长序列泛化
